About Shortly

Shortly is a modern, lightweight URL shortener built with FastAPI and Redis. It helps you shorten, manage, and track links effortlessly — designed for speed, simplicity, and reliability.

⚡

Blazing Fast

Optimized with Redis caching and async APIs for sub-millisecond performance.

🔗

Simple & Reliable

Shorten links instantly — no ads, no bloat, just clean analytics and control.

🚀

Built for Developers

Full REST API with Upstash Redis, perfect for automation and integrations.

Made by Sameer

Inspired by Bitly. Built with ❤️ using FastAPI, Tailwind, and Redis Cloud.
